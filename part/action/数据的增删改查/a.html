字符串操作

toUpperCase,toLowerCase

字符串全部大小写
string.toUpperCase()

indexOf()

索引指定字符下标(无返回-1)
string.index("abc")

substring(start,end)

获取从开始下标到结束下标前的字符(不包含结束下标的字符,0,5只截取前五个字符)
string.substring(0,5),string.substring(5)
var section = myString.substr(6,4); //结果: "good"

substr(start,num)

从开始下标截取num个字符返回
string.substr(0,5)

replace('a','b')

搜索指定字符串a替换为b
string.replace('a','b')

split()

字符串分割
string.split('a')

trim()

清理字符串两侧空格(返回清除后的字符串，原字符串不变)
string.trim()

数组操作Array

indexOf()

索引指定字符下标(无返回-1)
array.indexOf("abc")

slice(start,end)

获取从开始下标到结束下标前的数组(不包含结束下标的数组,0,5只截取前五个数组)
array.slice(0,5),array.slice(5),array.slice()//复制数组

push和pop

数组最后一位的添加与删除(前者返回数组长度,后者返回被删除的元素)
array.push("a","b"),array.pop()

unshift和shift

数组开头的添加与删除(前者返回数组长度,后者返回删除后的数组)
array.push("a","b"),array.pop()

sort

对数组进行排序(默认为字符编码顺序,可自行设置)
array.sort()

reverse

数组反序
array.reverse()

splice(起始下标,删除个数,添加元素,添加元素,...)

修改数组用
var arr = ['a','b','c','d','e','f'];
arr.splice(2,3,'1','2')//返回值为三个被删除的元素'c','d','e'
arr ['a', 'b', '1', '2', 'f']
arr.splice(2,2)仅删除//返回被删除元素
arr ['a', 'b', 'f']
arr.splice(2,0,'1','2')仅添加
arr ['a', 'b', '1', '2', 'f']

concat

数组拼接
将多个数组拼接形成一个新数组
arr=[a,b,c];var b = arr.concat([1,2,3])//原arr并未发生改变

join

将数组按指定符号连接形成字符串
arr.join("-")

对象操作object

for(var prop in obj){ if(obj.hasOwnProperty(prop)){ console.log(obj[prop]); } }

foreach

a.forEach(function (element, index, array) { // element: 指向当前元素的值 // index: 指向当前索引 // array: 指向Array对象本身 console.log(element + ', index = ' + index); });

es6新数据格式Map 和 Set

Map 对值

var m = new Map();
m.set('Adam', 67); // 添加新的key-value
m.set('Bob', 59);
m.has('Adam'); // 是否存在key 'Adam': true
m.get('Adam'); // 67
m.delete('Adam'); // 删除key 'Adam'
m.get('Adam'); // undefined

Set 无重复元素数组

var s1 = new Set()
add添加
delete删除
